1. Introduction to CSS & Why It Is Important CSS (Cascading Style Sheets) ek styling language hai jo HTML elements ko design karne ke liye use hoti hai. CSS ki Importance Website ko attractive banati hai. Colors, fonts aur layouts control karti hai. Responsive design create karne mein help karti hai. Code ko maintain karna easy ho jata hai. User experience improve hota hai. Without CSS: Website sirf plain text ki tarah dikhegi. With CSS: Website professional aur beautiful dikhegi. 2. Styling Text Using CSS A. font-family Text ka font change karta hai. CSS p{ font-family: Arial, sans-serif; } B. font-style Text ko italic ya normal banata hai. CSS p{ font-style: italic; } Values: normal italic oblique C. font-weight Text ki boldness control karta hai. CSS p{ font-weight: bold; } Values: normal bold 100–900 D. line-height Lines ke beech spacing set karta hai. CSS p{ line-height: 30px; } E. text-decoration Text ke niche line ya decoration lagata hai. CSS a{ text-decoration: none; } Values: none underline overline line-through F. text-align Text ki alignment set karta hai. CSS p{ text-align: center; } Values: left center right justify G. text-transform Text ka case change karta hai. CSS p{ text-transform: uppercase; } Values: uppercase lowercase capitalize H. letter-spacing Letters ke beech gap badhata hai. CSS h1{ letter-spacing: 3px; } I. word-spacing Words ke beech gap badhata hai. CSS p{ word-spacing: 10px; } J. text-shadow Text par shadow add karta hai. CSS h1{ text-shadow: 2px 2px 5px gray; } Syntax CSS text-shadow: horizontal vertical blur color; Example CSS h1{ text-shadow: 3px 3px 8px red;
